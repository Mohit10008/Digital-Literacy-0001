## Project Overview
This repository is the final submission for my **Digital Literacy (CSE0001)** project at **VIT Bhopal University**. As a **Student Digital Ambassador**, I have developed a series of resources to help my peers navigate the digital world safely, responsibly, and professionally. 

The project covers five core modules, ranging from visual communication to cybercrime awareness.

---

## Module Summaries

### Task 1: Digital Literacy Awareness Infographic
I designed a visual "Digital Literacy Essentials" guide using Canva to help students understand the definition of digital literacy, safe internet habits, and professional online branding.
* **Folder:** `Task 1 Presentation/`

### Task 2: Student Digital Portfolio
I established a professional foundation on **GitHub**, **LinkedIn**, and **Kaggle**. This task involved setting up a custom profile README and updating my academic details to build a long-term professional brand.
* **Folder:** `Task 2 Portfolio/`

### Task 3: Coding & Collaboration Platforms
I completed a Python coding challenge on **HackerRank** to practice technical logic. I also developed a 5-question **Digital Literacy Awareness Quiz** using Google Forms to engage with my batchmates.
* **Quiz Link:** https://docs.google.com/forms/d/e/1FAIpQLSc7ovbylFyh-OKdr03sneMLM9KvtqEFo3GAU-SSBktfV8oscw/viewform?usp=publish-editor
* **Folder:** `task-3-platforms/`

### Task 4: Professional Communication & Etiquette
This module focuses on professional email drafting and social media responsibility. It includes sample emails for academic and internship requests, along with a "Do's and Don'ts" guide for college students.
* **Folder:** `Task 4 Email Etiquette/`

### Task 5: Cybercrime Awareness & Prevention
I conducted a case study on **UPI/Online Payment Fraud** and compiled an 8-tip "Stay Safe Online" prevention checklist to help peers identify and report cyber threats.
* **Folder:** `Task 5 Cybercrime/`
